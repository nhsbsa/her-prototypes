Trust administrator dashboard

Set up, edit or disable a hospital trust from the NHS Report a Fault service.

Before you start adding a trust

You will need:

  • the trust name
  • the email domains that will be allowed to report faults
  • email addresses for trust staff who will access the dashboard
  • the phone number and extension for the trust helpdesk
  • details of how fault locations will be entered at the trust
  • an email contact for the facilities management company responsible for the trust
