Location Options

Each Trust has it's own method to specify a location, on site.

Some Trusts require only one value, such as room number. Others need several values.

This section allows each Trust to have a custom configuration to suit its needs when asking for a location description:

  • The first field is required, it is always shown to the reporter and must be filled in. It's label can be changed to suit the Trust, e.g. 'room number', 'PFI number', etc.
  • The next 4 fields, ‘location input’ (1-4), are optional and each one can be configured with:
  1. Label: Specify the information needed from the reporter, e.g. ‘Building Name’, ‘Floor number’, etc.
  2. Required: Tick this box if you require the reporter to add this information. Leave blank if the information is optional.
